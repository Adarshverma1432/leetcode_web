Sum of Multiples Solution

Problem Description:

Given an integer n, return the sum of all numbers from 1 to n (inclusive) that are divisible by 3, 5, or 7.

Example:
Input: n = 10
Output: 40

Explanation: Numbers divisible by 3, 5, or 7 are → 3, 5, 6, 7, 9, 10
Sum = 3 + 5 + 6 + 7 + 9 + 10 = 40

Approach:
We loop from 1 to n and check:
If a number is divisible by 3, 5, or 7 → add it to sum.

Condition:
i % 3 == 0 || i % 5 == 0 || i % 7 == 0 
class Solution {
    public int sumOfMultiples(int n) {

        int sum = 0;

        for (int i = 1; i <= n; i++) {

            if (i % 3 == 0 || i % 5 == 0 || i % 7 == 0) {
                sum += i;
            }

        }

        return sum;
    }
}

🌳 Trees Complete Guide

📌 What is Tree?

Tree ek hierarchical data structure hai jisme nodes parent-child relation me connected hote hain. Top node ko root kehte hain.

1 / \ 2 3 / \ 4 5

Root → 1

📌 Important Terms

  • Root → sabse upar node
  • Parent → jiske niche child ho
  • Child → jo parent ke niche ho
  • Leaf → jiske koi child na ho
  • Height → max depth

📌 Types of Trees

1. Binary Tree

Har node ke max 2 children hote hain

2. Binary Search Tree (BST)

Left < Root < Right

3. Balanced Tree

Height minimum hoti hai

📌 Binary Tree

10 / \ 5 15

📌 Binary Search Tree (BST)

10 / \ 5 20 / \ 2 7

Left < Root < Right

📌 Tree Traversals 🔥

1. Inorder (Left → Root → Right)

2. Preorder (Root → Left → Right)

3. Postorder (Left → Right → Root)

📌 Traversal Example

1 / \ 2 3

Inorder → 2,1,3

Preorder → 1,2,3

Postorder → 2,3,1

📌 Time Complexity

  • Traversal → O(n)
  • Search (BST) → O(log n)
  • Insert → O(log n)

📌 Important Patterns 🔥

1. DFS (Depth First Search)

Deep jao → recursion use hota hai

2. BFS (Level Order)

Level by level traversal

📌 DFS vs BFS

DFS → recursion / stack

BFS → queue use hota hai

💻 Basic Code (Traversal)

void inorder(Node root){ if(root == null) return; inorder(root.left); System.out.print(root.val); inorder(root.right); }

🎯 Important Questions

  • Maximum Depth of Tree
  • Invert Binary Tree
  • Same Tree
  • Level Order Traversal

🧠 Dynamic Programming (DP)

📌 What is DP?

DP ek optimization technique hai jisme repeated problems ko store karke solve kiya jata hai.

📌 Core Idea

1. Overlapping Subproblems

Same problem baar-baar solve ho raha hai

2. Optimal Substructure

Chhote solution se bada solution banta hai

📌 Types of DP

1. Memoization (Top-Down)

Recursion + store results

2. Tabulation (Bottom-Up)

Loop use karke build karte hain

📌 Fibonacci Example

F(5) → F(4) + F(3)
Repeated calls hoti hain

📌 DP Table Concept

📌 How to Solve DP

  • Problem ko small parts me todo
  • Recurrence relation banao
  • Base case define karo
  • Store results

📌 Time Complexity

DP recursion ko optimize karta hai → exponential se O(n) ya O(n²)

📌 Important Patterns 🔥

1. 1D DP

Fibonacci, Climbing Stairs

2. 2D DP

Grid problems

3. Knapsack

Weight/value optimization

💻 Basic Code

int[] dp = new int[n+1]; dp[0] = 0; dp[1] = 1; for(int i=2;i<=n;i++){ dp[i] = dp[i-1] + dp[i-2]; }
